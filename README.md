## 🌟 Overview

**Velocity** is a high-fidelity, professional-grade financial dashboard designed for modern asset monitoring and analytics.

It combines:

* ⚛️ React for UI & state management
* 🎞️ GSAP for cinematic animations
* 📊 Recharts for data visualization

---

## ✨ Key Features

### 🎬 Cinematic UI/UX

* GSAP-powered smooth animations
* Interactive hover effects
* Glassmorphism dark UI design

---

### 📊 Data Visualization

* KPI cards with animated counters
* Area & Bar charts (Income vs Expense)
* Asset allocation (Pie chart)
* Market watch with sparkline charts

---

### 🔐 Role-Based UI (Simulation)

* Admin mode → Add/Delete transactions
* Viewer mode → Read-only access
* Toggle role dynamically

---

### 📋 Management System

* Transaction tracking
* Search & filtering
* Financial analytics
* Performance insights

## 📦 Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone <your-repository-url>
cd velocity-dashboard
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Run Project

```bash
npm run dev
```
